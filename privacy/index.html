2. General information on data processing

The protection of your personal data is important to me. This privacy policy informs you about the personal data collected when you visit this website and the purposes for which it is processed.

3. Hosting

This website is hosted on GitHub Pages, a service provided by GitHub, Inc. (88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA), a subsidiary of Microsoft Corporation. When you access this website, the hosting provider automatically collects information in server log files that your browser transmits automatically. This includes:

  • IP address of the requesting device
  • Date and time of the request
  • Name and URL of the retrieved file
  • Volume of data transferred
  • Browser type and version
  • Operating system used
  • Referrer URL (previously visited page)

This data is processed to ensure a smooth connection, to guarantee system security, and for administrative purposes. The legal basis is Art. 6(1)(f) GDPR (legitimate interest).

The retention period for server log files is governed by the policies of GitHub, Inc. and, according to the provider, is typically no longer than 30 days, unless longer storage is necessary in individual cases to defend against security incidents. For details, see the GitHub Privacy Statement.

Provider: GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA. GitHub is certified under the EU-U.S. Data Privacy Framework. More information: GitHub Privacy Statement.

4. Local storage for user preferences

This website stores only one preference that you have actively set — your choice between light and dark appearance (“theme”) — in a localStorage entry named bp-theme on your device. This storage does not require consent under § 25(2) no. 2 TDDDG (German Telecommunications-Digital Services-Data Protection Act) because it is strictly necessary to provide the feature you have explicitly requested. The setting is not transmitted to me or to any third party. You can delete the entry at any time through your browser settings.

5. No cookies, no tracking, no social media plugins

Beyond the local storage described in section 4, this website does not use cookies, analytics or tracking tools, or social media plugins. No personal data about your usage behavior is collected.

6. Contact

If you contact me by email, your details including your email address will be stored for the purpose of processing the inquiry and in case of follow-up questions. This data will not be shared with third parties. The legal basis is Art. 6(1)(f) GDPR, or Art. 6(1)(b) GDPR if the inquiry is directed at the conclusion of a contract.
